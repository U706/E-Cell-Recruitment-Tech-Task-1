# FocusFlow - Startup Landing Page

A clean, modern, and responsive startup landing page built using **HTML, CSS, and JavaScript**.  
This project presents a fictional startup called **FocusFlow**, a productivity tool designed to help users manage tasks and stay focused.

##  Project Overview

FocusFlow is a simple yet visually appealing landing page that demonstrates how a startup can present its idea effectively.

The page includes:
- A strong **hero section** with a clear call-to-action
- Structured sections explaining the **problem and solution**
- A responsive layout suitable for all devices
- A working **contact form**


##  Technologies Used

- **HTML5** → Structure of the webpage  
- **CSS3** → Styling, layout, responsiveness, animations  
- **JavaScript (Vanilla JS)** → Interactivity and form handling
